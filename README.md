# Playable

Production-ready playable ads. Built with TypeScript and Vite.

![Playable Hero](https://raw.githubusercontent.com/heyrtl/playable/main/hero.png)

## Live Demos

Visit [heyrtl.github.io/playable](https://heyrtl.github.io/playable) to interact with all demos.

## Featured

- **Frost World** - 4X Resource management and survival mechanics (258kb)
- **Bubble Shooter** - Classic arcade bubble-popping (151kb)
- **Block Blast** - Strategic block placement puzzle (299kb)
- **LabSort** - Laboratory color water sorting mechanics (100kb)

## Additional Demos

- Gem Matcher - Match-3 puzzle (176kb)
- Dragon Merge - Merge-2 progression game (241kb)
- Cookie Empire - Idle clicker (188kb)
- Word Match - Bilingual word puzzle (198kb)

## Tech Stack

- React & TypeScript
- Vite build system
- Tailwind CSS
- MRAID v2.0 compliant
- Single-file HTML output
- Zero runtime dependencies

All playables are optimized for <300ms load time on 3G networks and universal ad network compatibility.

## About

Part of [TouchPlay](https://touchplay.dev) - building tools for faster playable ad production.
